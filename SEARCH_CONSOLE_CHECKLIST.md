# Search Console Checklist

## Before Requesting Indexing
- production domain resolves to Vercel
- HTTPS works on apex and `www`
- `robots.txt` is reachable
- `sitemap.xml` is reachable
- canonical host is `https://schnellsicherumzug.de`
- no accidental `noindex` on public service pages

## Submit
- verify domain property: `schnellsicherumzug.de`
- submit sitemap: `https://schnellsicherumzug.de/sitemap.xml`

## Inspect
- homepage
- `/umzug`
- `/entsorgung`
- `/montage`
- `/preise`
- `/booking`

## QA
- mobile-friendly test
- rich results test
- page speed checks
- broken link checks
